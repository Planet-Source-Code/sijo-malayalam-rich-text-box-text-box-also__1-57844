Place

SIJOSoftCtrl.exe  - In System Directory
The Dynamic Link Library that guids the OCX (like Richedit32.dll)

SIJOSoftTrans.dll - In System Directory
The English to Malayalam Transulation Engine

SIJOSoft.ttf - In System Directory and Font Directory
The Greate Malayalam Font
this font must be in system directory(or with dynamic library +engine)



To run this application you must have;

	1) SSMLM.OCX
           (Same to Richtext.ocx)
	2) SIJO Soft Malayalam SDK must be installed on this computer
           (Same to Richedit32.dll)

There is a Malayalam Notepad developed with this control
Can be found at 

http://sijo.8k.com/ssmlm.htm
